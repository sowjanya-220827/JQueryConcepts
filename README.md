# jQuery Concepts Project

## Overview

This project demonstrates fundamental **jQuery concepts** through simple and interactive examples.
It includes examples of selectors, attribute methods, events, effects, style manipulation, and DOM traversing.

The project is designed for **learning and academic purposes**, helping students understand how jQuery interacts with HTML elements and the DOM.

## Topics Covered

### 1. jQuery Selectors

Examples include:

* Basic Selectors

  * `$("element")`
  * `$("#id")`
  * `$(".class")`
  * `$("element, element")`

* Hierarchy Selectors

  * `$("parent child")`
  * `$("parent > child")`
  * `$("prev + next")`
  * `$("prev ~ siblings")`

* Filter Selectors

  * `:first`
  * `:last`
  * `:eq()`
  * `:even`
  * `:odd`
  * `:not()`

* Attribute Selectors

  * `[attribute]`
  * `[attribute=value]`
  * `[attribute^=value]`
  * `[attribute$=value]`
  * `[attribute*=value]`
